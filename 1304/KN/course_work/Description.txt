﻿Вдохновляемся к реализации устройством реальной человеческой армии. А именно устройством стрелкового взвода. Подразделение должно представлять собой единую мобильную единицу, поэтому все его участники должны иметь примерно одну скорость. Наш моделируемый взвод будет укомплектован наиболее простым образом:
* Командир взвода - 1 штука
* Стрелок - N штук
Каждый солдат:
* Имеет порядковый номер, это позволит нам организовать их в шеренги и походные строи по командам командира
* Находится в одном из состояний исполнения действия:
   * СОВЕРШАЮ ДВИЖЕНИЕ (
        ID ПОЗИЦИИ : String,
        СКОРОСТЬ : double)
   * ПРОИЗВОЖУ ОГОНЬ (НАПРАВЛЕНИЕ : double)
   * Имеет значение здоровья (0 - 100%)


        Солдаты производят команды в эфир союзникам:
   * ПРИКАЗ (ИСТОЧНИК : String)
   * ПОСТРОЕНИЕ (
        КОЛ-ВО ШЕРЕНГ : int,
        ID ПОЗИЦИИ : String,
        НАПРАВЛЕНИЕ : double)
   * МАРШ (КОЛ-ВО ШЕРЕНГ : int, ID ПОЗИЦИИ : int)
   * ОГОНЬ (НАПРАВЛЕНИЕ : double)
      * ГОТОВ (ИСТОЧНИК : String)
      * ВИЖУ ПРОТИВНИКА (ID ПОЗИЦИИ : String)
      * РАНЕН (ИСТОЧНИК : String)


Механика стрельбы:
Когда солдат производит огонь, он публикует информацию о выстреле как структуру (ID КОМАНДЫ : int, POS_X : double, POS_Y : double, НАПРАВЛЕНИЕ : double) в эфир выстрелов. Высшая игровая сущность - Арена - слушает этот эфир и наносит урон роботу, если тот на пути поражения.


Первичная структура нод и топиков:


Задача стрелков:
      * Правильно реагировать на (выполнять) приказы
      * Оповещать о выполнении приказа
      * Оповещать об обнаруженном противнике
      * Оповещать о выходе из боя (РАНЕН)
Задача командира:
      * Управлять подразделением в режиме разведки до появления первых сведений о противнике (МАРШ в N шеренг на точку КОМАНДИР_BACK)
      * Управлять подразделением: выдвигать его на боевую позицию (вычислять её из информации о противниках) и давать приказы по ведению огня при наличии информации о противнике
      * Вести огонь по противнику


Механика построений и перемещений


        Каждый солдать при получении приказа знает на кого ориентироваться
