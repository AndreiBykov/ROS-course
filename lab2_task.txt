Разработать программу, которая демонстрирует систему связанных между собой стержней (количество произвольное). каждый стержень прикреплён к обоими концами к соседним стержням. Самый первый берёт начало из точки (0,0,0). Каждый стержень вращается вокруг точки, которой он прикреплён к предыдущему стержню. Вращение каждого стержня не совпадает ни по скорости, ни по плоскости вращения ни с одним другим стержнем.
Математические вычисления положения точек стержня должны быть реализованы при помощи пакета tf.
Визуализация системы должна быть выполнена в rviz
